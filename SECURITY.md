# Security Policy

## Supported Versions

Security fixes are provided for the latest minor release on the `main` branch.

## Reporting a Vulnerability

Please do not open public GitHub issues for security-sensitive reports.

Report privately through one of these channels:

- GitHub Security Advisories: `https://github.com/Biglone/session-bridge/security/advisories`
- Email: `biglone@users.noreply.github.com`

Include the following details:

- Affected version and environment
- Reproduction steps or proof of concept
- Impact and expected severity
- Any suggested remediation

You will receive an acknowledgment within 3 business days.
